# Clinic Management Backend

Backend API for clinic management system using Node.js and Express.

## Installation

Clone project

git clone https://github.com/your-repo/clinic-backend.git

cd clinic-backend

Install dependencies

npm install

Create environment file

cp .env.example .env

Run server

npm run dev

Server runs at

http://localhost:3000
